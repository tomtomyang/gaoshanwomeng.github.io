1.安装hugo

  1. 在hugo的git仓库下载最新版本

    ubuntu系统选择hugo_0.71.1_Linux-64bit.deb

  2. 安装完成后在命令行进行验证

1
2
3

hugo version
Hugo Static Site Generator v0.71.1-A301F6B2 linux/amd64 BuildDate: 2020-05-25T09:09:07Z
如果显示版本信息说明安装成功
  1. 初始化hugo博客，生成blog文件夹，进入blog文件夹
1
2

hugo new site blog
cd blog
  1. blog文件夹的目录结构如下：

├─archetypes

├─themes 主题文件夹
└─config.toml blog的配置文件

3.部署到服务器

这里以免费的gitee pages为例，github pages，gitlab pages都是一样的操作。

1. 推送到远端仓库

将public文件夹里的全部内容push到gitee仓库

1
2
3
4
5
6
7

cd public
git init
git add *
git commit -m "blog first commit"
git remote add origin 仓库地址
git push -u origin master
会要求输入gitee的账号和密码

2. 开启gitee pages服务

3. 访问页面

打开gitee page 生成的地址，就可以看到博客已经部署到服务器上

4. 可能遇到的问题

如果出现页面的css样式没有加载的情况，需要在blog目录下新建一个_config.yml文件进行配置

touch __config.yml

添加以下内容
theme: npq-hugo

hugo -D
cd public
git add *
git commit -m "second commit"
git push -u origin master

打开gitee page配置页面，点击更新
重新访问博客，css样式加载成功
