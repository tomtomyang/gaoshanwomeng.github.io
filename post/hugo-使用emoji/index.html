hugo 使用emoji​​
📅 2020-05-29 👦 Tomtom Young 📖 271字 ⏱ 1分钟

自己搭建的博客一直不能显示emoji，现在记录一下在hugo博客中怎么开启emoji支持。

1. config.toml配置

在config.toml中进行配置

1enableEmoji = true

2. 添加CSS样式

在网页的CSS文件中加入以下内容：

1.emoji {
2	font-family: Apple Color Emoji,Segoe UI Emoji,NotoColorEmoji,Segoe UI Symbol,Android Emoji,EmojiSymbols;
3}

3. 验证一下
