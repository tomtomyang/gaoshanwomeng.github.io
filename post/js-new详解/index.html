js中关键字new的原理（含实现代码）

1. new

new会实例化类的实例对象，或者说，使用构造函数，构造一个我们想要的对象。

关于类的定义有两种，使用构造函数定义的类，使用Class关键字定义的类。

1. 构造函数定义类

在MDN中，其实找不到关于构造函数的准确定义，甚至可以说没有构造函数这中说法，唯一和构造有关的定义是构造方法，指的是class中的constructor()，但是我们这里讨论的是，使用new关键字调用，实现构造对象功能的函数。

在 JavaScript 中，用 new 关键字来调用的函数，称为构造函数。构造函数首字母一般大写。

可以这样定义，使用new关键字，执行构造对象功能的函数，统一称为构造函数。

2. Class定义类

关于Class的详细说明，可以看《js Class详解》。

3. new 的流程

new 关键字会进行如下的操作：

  1. 创建一个空的对象（即{ }）；
  2. 给这个空对象添加属性__proto__，将该属性链接为构造函数的原型对象prototype ；
  3. 构造函数内部的this被赋值为这个新对象（即this指向新对象）；
  4. 执行构造函数内部的代码（即给新对象添加属性）；
  5. 如果构造函数返回非空对象，则返回该对象；否则，返回刚创建的新对象；

用代码来说就是这样一个过程：

1var obj = {};
2obj.__proto__ = Animal.prototype;
3var result = A.call(obj);
4return typeof result === 'object'? result : obj;
