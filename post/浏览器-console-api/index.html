console.assert(expression, object)

级别：Error

expression评估为false时，将object写入控制台。

1const x = 5;
2const y = 3;
3const reason = 'x is expected to be less than y';
4console.assert(x < y, {x, y, reason});

console.clear()

清空控制台

1console.clear();

如果启用了Preserve Log，console.clear()会被禁用。

console.debug()

级别：Verbose

console.log(object [, object, ...\])除不同的日志级别外，其他均相同。

1console.debug('debug');

console.dir(object)

级别：Info

打印指定对象的JSON表示形式。

1console.dir(document.head);

console.dirxml(node)

级别：Info

打印的后代的XML表示形式node。

1console.dirxml(document);

console.error()

级别：Error

打印object到控制台，将其格式化为错误，并包括堆栈跟踪。

1console.error("I'm sorry, Dave. I'm afraid I can't do that.");

console.info()

级别：Info

与相同console.log(object [, object, ...\])。

1console.info('info');

console.log()

级别：Info

将消息打印到控制台。

1console.log('log');

console.table(array)

级别：Info

将对象数组记录为表格。

 1console.table([
 2  {
 3    first: 'René',
 4    last: 'Magritte',
 5  },

 9    birthday: '18930113',
10  },
11  {
12    first: 'Henri',
13    last: 'Matisse',
14  }
15]);

console.time([label])

启动一个新的计时器。呼叫console.timeEnd([label])以停止计时器并将经过的时间打印到控制台。

1console.time();
2for (var i = 0; i < 100000; i++) {
3  let square = i ** 2;
4}
5console.timeEnd();

console.timeEnd([label])

级别：Info

console.warn()

级别：Warning

将警告打印到控制台。

1console.warn('warn');
