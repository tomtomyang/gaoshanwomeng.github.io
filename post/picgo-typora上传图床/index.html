1. 安装picgo-core

直接在typora里面安装picgo-core，选择下载或更新。

2. 配置picgo-core

打开配置文件，在配置文件里填入以下内容：

{
    "picBed": {
      "uploader": "tcyun",
      "tcyun": {
        "secretId": "",
        "secretKey": "",
        "bucket": "", 
        "appId": "",
        "area": "", 
        "path": "",
        "customUrl": "",
        "version": "v5"
      }
    },
    "picgoPlugins": {} 
  }

其中的secretId，secretKey在这里查看：

https://console.cloud.tencent.com/cam/capi

bucket和appid和area需要在控制台里查看：

https://console.cloud.tencent.com/cos5/bucket

3. 验证是否成功

选择验证图片上传选项

显示验证成功说明picgo-core设置成功，markdown里再插入图片就不需要手动操作，大大提高了效率👍。
