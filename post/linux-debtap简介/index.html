1.什么是debtap

debtap是一个将.deb包转化为.pkg.tar.zst的工具，从而实现了在arch上安装debian的软件。由于ubuntu的盛行，很多软件都会有专门的.deb安装包，我们可以借助debtap来安装他们。

2.安装debtap

在arch系上安装需要执行以下命令：

1
2

yay -S debtap
sudo debtap -u

sudo debtap -u必须要执行，这应该是类似于初始化的操作，要下载一下debtap需要的包。

如果sudo debtap -u执行很慢的话，可以换源：

打开 /usr/bin/debtap，更换 debtap内容：

4
5

替换：http://archive.ubuntu.com/ubuntu/dists
https://mirrors.ustc.edu.cn/ubuntu/dists/

3.使用debtap

这里以listen1为例。

首先下载到listen1的deb包：

然后执行：

1

sudo debtap Listen1_2.9.0_linux_amd64.deb

开始重新打包，过程中会询问新的包的名字，还有使用的开源协议，这里名字随便起，开源协议输入gui即可，还有一步需要选择用什么编辑器打开一个文件，具体是什么文件我给忘记了，我选择的是使用默认编辑器，然后提示没有设定默认编辑器，这一步被跳过。

再等待一会就会显示打包成功，在同级目录下得到一个新的文件：

使用

然后执行：

1

sudo pacman -U listen1-2.9.0-1-x86_64.pkg.tar.zst

就可以安装成功了！
