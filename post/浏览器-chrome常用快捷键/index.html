浏览器 chrome常用快捷键
📅 2021-05-10 👦 Tomtom Young 📖 279字 ⏱ 1分钟

1. 浏览器快捷键

  • f12/ctrl+shift+c 打开devtools element界面
  • ctrl+shift+j 直接打开devtools console界面
  • ctrl+n 打开新的浏览器窗口
  • ctrl+shift+n 打开新的无痕浏览窗口
  • ctrl+w 关闭当前页面
  • shift+esc 打开任务管理器
  • ctrl+1-x 切换到不同页面
  • ctrl+p 打印页面

2. 开发者工具快捷键

  • ctrl+[/] 切换上一个/下一个面板
  • ctrl+shift+c 抓屏元素

2.1 console

  • ctrl+l 清除控制台
  • shift+enter 多行输入

2.2 elements

  • h 隐藏元素
  • f2 以html编辑
  • ctrl+alt+点击箭头 展开/折叠节点及其所有子节点
2.2.1 style
  • shift+↑/↓ 以10为增量增大减小
  • alt++↑/↓ 以0.1为增量增大减小
  • ctrl + 点击属性 转到源中样式规则属性声明行
  • ctrl + 点击属性值 转到源中属性值声明行

2.3 sources

  • ctrl+enter 运行代码段
  • ctrl+g 跳转到指定行
  • ctrl+m 跳转到匹配的括号
