1.了解vim

vim是最好的编辑之一，拥有以下几个优点：

  1. 高效率移动，使用vim的命令模式，可以让光标到达你想要的任何地方
  2. 高效率输入
    • vim可以进行关键词自动提示，输入几个字母然后按住ctrl+n，可以自动补全
    • vim拥有各种各样的插入命令，可以实现各种各样的输入
    • 多重剪贴板和各种各样的快捷键，运用熟练的话可以极大的提高编辑速度
  3. 使用vim的时候指运如飞，让小白和刚入门的同学感觉你很屌

2.vim的基本使用

1. vim的模式

编辑模式：

  • i：在当前字符前插入。

  • I：在光标所在行的行首插入。

  • a：在当前字符后插入。

  • A：在光标所在行的行尾插入。

  • o：在当前行的下一行插入新的一行。

  • O：在当前行的上一行插入新的 一行。

命令模式：

  • /word：在光标之后查找一个字符串word，按n向后继续搜索，shift+n向上搜索。

  • ?word：在光标之前查找一个字符串word，按n向后继续搜索。搜索出来的字符串都会高亮显示，若想不高亮，输入:nohl。

  • :w 保存文本。

  • :q 退出vim。

  • :w! 强制保存，在root用户下，即使文本只读也可以完成保存。

  • :q! 强制退出，所有改动不生效。

  • :wq 保存退出。

  • :x 类似于wq，更改了文件以后，wq和x的作用是一样的，若没有更改文件，使用wq，文件的mtime会改变，而x不会

3.安装Vunble

vunble是vim的插件管理工具。

  1. 在.vim文件夹下新建bundle文件夹，并将Vunble插件clone到里面

mkdir .vim/bundle
git clone https://github.com/VundleVim/Vundle.vim.git .vim/bundle
  1. 在.vimrc里添加以下内容：

set nocompatible 

set rtp+=~/.vim/bundle/Vundle.vim
call vundle#begin()

Plugin 'gmarik/Vundle.vim'

call vundle#end()
filetype plugin indent on
  1. 打开vim编辑器，在命令模式下输入:PluginInstall安装Vunble，提示Done说明安装完成。

set nocompatible

Plugin 'gmarik/Vundle.vim'
Plugin 'scrooloose/nerdtree'

call vundle#end()
filetype plugin indent on

" NerdTree插件配置
map <f2> :NERDTreeToggle<cr>
let g:NERDTreeDirArrowExpandable = '+'
let g:NERDTreeDirArrowCollapsible = '-'
let g:NERDTreeWinPos='left'
let g:NERDTreeSize=30
let g:NERDTreeShowLineNumbers=1
let g:NERDTreeHidden=1
  1. 再次执行:PluginInstall，提示Done说明安装完成。
  2. F2打开和关闭文件目录，ctril+ww进行两个窗口的切换

  1. 其他插件的安装 和nerdtree基本一样，都需要在.vimrc里进行配置，然后执行:PluginInstall命令。
