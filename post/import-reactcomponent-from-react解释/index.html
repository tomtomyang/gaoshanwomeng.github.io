在学习react框架时，我们会在App.js中写这样一种语法：

1import React,{Component} from 'react'

表面看起来是结构赋值的样子，但其实，这是ES6的另一种引入方式，即default引入与单个引入相结合的方式。

想使用这种引入方式，就要求在暴露模块的文件中，既使用了export default 语法进行导出，也使用了export语法进行了单独的导出。

类似于下面这种写法：

1const React = { a:1, b: 2}
2
3export class Component {
4}
5
6React.Component = Component
7
8export default React
