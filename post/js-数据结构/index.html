3. 链表

链表:存贮有序元素的集合, 但是不同于数组,每个元素是一个存贮元素本身的节点和指向下一个元素引用组成 要想访问链表中间的元素,需要从起点开始遍历找到所需元素

  1class Node {
  2    constructor(element) {
  3      this.element = element;
  4      this.next = null;
  5    }
  6  }
  7
  8  // 链表
  9  class LinkedList {
 10    constructor() {
 11      this.head = null;
 12      this.length = 0;
 13    }
 14
 15    // 追加元素
 16    append(element) {
 17      const node = new Node(element);
 18      let current = null;
 19      if (this.head === null) {
 20        this.head = node;
 21      } else {
 22        current = this.head;
 23        while (current.next) {
 24          current = current.next;
 25        }
 26        current.next = node;
 27      }
 28      this.length++;
 29    }
 30
 31    // 任意位置插入元素
 32    insert(position, element) {
 33      if (position >= 0 && position <= this.length) {
 34        const node = new Node(element);
 35        let current = this.head;
 36        let previous = null;
 37        let index = 0;
 38        if (position === 0) {
 39          this.head = node;
 40        } else {
 41          while (index++ < position) {
 42            previous = current;
 43            current = current.next;
 44          }
 45          node.next = current;
 46          previous.next = node;
 47        }
 48        this.length++;
 49        return true;
 50      }
 51      return false;
 52    }
 53
 54    // 移除指定位置元素
 55    removeAt(position) {
 56      // 检查越界值
 57      if (position > -1 && position < length) {
 58        let current = this.head;
 59        let previous = null;
 60        let index = 0;
 61        if (position === 0) {
 62          this.head = current.next;
 63        } else {
 64          while (index++ < position) {
 65            previous = current;
 66            current = current.next;
 67          }
 68          previous.next = current.next;
 69        }
 70        this.length--;
 71        return current.element;
 72      }
 73      return null;
 74    }
 75
 76    // 寻找元素下标
 77    findIndex(element) {
 78      let current = this.head;
 79      let index = -1;
 80      while (current) {
 81        if (element === current.element) {
 82          return index + 1;
 83        }
 84        index++;
 85        current = current.next;
 86      }
 87      return -1;
 88    }
 89
 90    // 删除指定文档
 91    remove(element) {
 92      const index = this.indexOf(element);
 93      return this.removeAt(index);
 94    }
 95
 96    isEmpty() {
 97      return !this.length;
 98    }
 99
100    size() {
101      return this.length;
102    }
103
104    // 转为字符串
105    toString() {
106      let current = this.head;
107      let string = "";
108      while (current) {
109        string += ` ${current.element}`;
110        current = current.next;
111      }
112      return string;
113    }
114  }

5. 树

Tree(树)数据结构是多层结构。与Array，Stack和Queue相比，它也是一种非线性数据结构。这种结构在插入和搜索操作时效率很高。我们来看看树型数据结构的一些概念。

  • root：树的根节点，无父节点
  • parent node：上层的直接节点，只有一个
  • child node：下层的直接节点可以有多个
  • siblings：共享同一个父节点
  • leaf：没有孩子的节点
  • Edge：节点之间的分支或链接
  • path：从起始节点到目标节点的边
  • Height of Nod：特定节点到叶节点的最长路径的边数
  • Height of Tree：根节点到叶节点的最长路径的边数
  • Depth of Node：从根节点到特定节点的边数
  • Degree of Node：子节点数

二叉树

特点：每个节点最多有两个子树的树结构

 1class NodeTree {
 2    constructor(key) {
 3      this.key = key;
 4      this.left = null;
 5      this.right = null;
 6    }
 7  }
 8
 9  class BinarySearchTree {
10    constructor() {
11      this.root = null;
12    }
13
14    insert(key) {
15      const newNode = new NodeTree(key);
16      const insertNode = (node, newNode) => {
17        if (newNode.key < node.key) {
18          if (node.left === null) {
19            node.left = newNode;
20          } else {
21            insertNode(node.left, newNode);
22          }
23        } else {
24          if (node.right === null) {
25            node.right = newNode;
26          } else {
27            insertNode(node.right, newNode);
28          }
29        }
30      };
31      if (!this.root) {
32        this.root = newNode;
33      } else {
34        insertNode(this.root, newNode);
35      }
36    }
37
38    //访问树节点的三种方式:中序,先序,后序
39    inOrderTraverse(callback) {
40      const inOrderTraverseNode = (node, callback) => {
41        if (node !== null) {
42          inOrderTraverseNode(node.left, callback);
43          callback(node.key);
44          inOrderTraverseNode(node.right, callback);
45        }
46      };
47      inOrderTraverseNode(this.root, callback);
48    }
49
50    min(node) {
51      const minNode = node => {
52        return node ? (node.left ? minNode(node.left) : node) : null;
53      };
54      return minNode(node || this.root);
55    }
56
57    max(node) {
58      const maxNode = node => {
59        return node ? (node.right ? maxNode(node.right) : node) : null;
60      };
61      return maxNode(node || this.root);
62    }
63  }

6. 图

Graph（有时称为网络）是指具有链接（或边）的节点集。根据联系是否有方向性，可以进一步分为两组（即定向图和不定向图）。Graph在我们的生活中被广泛使用——在导航应用中计算最佳路线，或者在社交媒体中推荐朋友，举两个例子。

图有两种表示形式：

邻接清单

在此方法中，我们在左侧列出所有可能的节点，并在右侧显示已连接的节点。

邻接矩阵

相邻矩阵以行和列的形式显示节点，行和列的交点诠释了节点之间的关系，0表示没有联系，1表示有联系，>1表示权重不同。

要查询图中的节点，必须用 “宽度优先搜索"（BFS）方法或 “深度优先搜索”（DFS）方法在整个树网中进行搜索。

让我们看一个例子的BFS在Javascript：

 1function bfs(graph, root) {
 2  var nodesLen = {};
 3  for (var i = 0; i < graph.length; i++) {
 4    nodesLen[i] = Infinity;
 5  }
 6  nodesLen[root] = 0;
 7  var queue = [root];
 8  var current;
 9  while (queue.length != 0) {
10    current = queue.shift();
11
12    var curConnected = graph[current];
13    var neighborIdx = [];
14    var idx = curConnected.indexOf(1);
15    while (idx != -1) {
16      neighborIdx.push(idx);
17      idx = curConnected.indexOf(1, idx + 1);
18    }
19    for (var j = 0; j < neighborIdx.length; j++) {
20      if (nodesLen[neighborIdx[j]] == Infinity) {
21        nodesLen[neighborIdx[j]] = nodesLen[current] + 1;
22        queue.push(neighborIdx[j]);
23      }
24    }
25  }
26  return nodesLen;
27}
