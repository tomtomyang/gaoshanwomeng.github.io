1. VueCLI 简介

vue-cli 俗称脚手架，可以快速创建 vue 项目结构，学习完 webpack 之后，我们会有一种体会就是项目的每个文件夹，每个文件都需要我们手动新建，小项目还好，如果遇到复杂的大项目，这会大大降低开发的效率。如果使用 vue-cli，我们可以快速生成 vue 项目，而不需要我们一点点的去创建/管理项目所需要的各种文件夹/文件。vue-cli 是对 webpack 的再封装，超级 webpack。

2. VueCLI 的优势

  1. 简单快速

使用 vue-cli，我们可以快速的新建一个 vue 项目，甚至可以使用图形化界面进行新项目的创建。

  1. 功能丰富

vue-cli 支持 Bable，Ts，ESLint，Axios 等。

  1. 易于扩展

vue-cli 有很多插件，可以实现大部分我们想要的功能。

3. 如何使用 VueCLI

1. 安装 VueCLI

1npm install -g @vue/cli
2vue -V

2. 创建 Vue 项目

1vue creat 项目名

执行这条命令后，会进入一个交互式的面板，会让你选择项目需要的插件，以及各种配置，具体配置根据项目和个人使用习惯而有所区别，我第一次创建项目使用的配置是：

  1. pick a preset –>manually select features
  2. check the features needed for your project –>bable，router，Linter
  3. use history mode for router –>n
  4. pick a linter config –>ESLint + Standard config
  5. pick addition lint features –>Lint to save
  6. where do you prefer placing config for … –>In dedicated config files
  7. save this as a present for future projects –>n

3. 配置项目

创建项目之后，我们可能会想要进行一些自定义配置，比如指定端口号，自动打开页面等，进行这些有两种方式，一种是在 package.json 中配置，另一种是新建 vue.config.js 进行配置，这里推荐第二种。

vue.config.js

1module.exports = {
2    devServer: {
3        //自动打开浏览器
4        open: true,
5        //指定端口号
6        port: 10010
7    }
8}

1cd 项目名
2npm run serve

5. 项目目录说明

  1. node_modules

项目依赖包。

  1. public

存放静态资源。

index.html：

是一个模板文件，作用是生成项目的入口文件。浏览器访问项目的时候就会默认打开的是生成好的 index.html。

  1. src 项目核心文件

  2. assets

  样式类文件，如 css、less、sass 以及外部的 js 文件都放在这里。
  1. components
  存放自定义组件的文件夹。
  1. App.vue
  根组件，所有页面都是在 App.vue 下进行切换的，也可以理解为所有的路由也是 App.vue 的子组件。
  1. main.js
  入口文件，主要作用是初始化 vue 实例，同时可以在此文件中引用某些组件库或者全局挂载一些变量。
  1. .babelrc

babel 配置参数。

  1. .gitignore

git 忽略文件。

  1. package-lock.json

是在 npm install 时候生成的一份文件，用于记录当前状态下实际安装的各个 npm package 的具体来源和版本号。

  1. package.json

模块基本信息项目开发所需要的模块，版本，项目名称。

  1. README.md 项目说明

  2. webpack.config.js webpack 配置文件

6. 组件化开发

1. 组件化开发

在使用 vue-cli 进行开发时，使用组件化开发方式，也就是.vue 文件。一个.vue 文件就是一个组件，里面包含了 html，js，css 三部分，通过这种方式，分离了每个组件之间的关联，提高了内聚，因此叫做组件化开发。

2. 新建组件

VueCLI 的组件都写在 src/components 里面，如果我们需要新建组件，可以在 webstorm 里直接右键选择新建 vue components，会自动生成 components 的结构，很快很方便。
