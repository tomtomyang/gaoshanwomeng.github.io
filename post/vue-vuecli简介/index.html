1.vue-cli简介

vue-cli俗称脚手架，可以快速创建vue项目结构，学习完webpack之后，我们会有一种体会就是项目的每个文件夹，每个文件都需要我们手动新建，小项目还好，如果遇到复杂的大项目，这会大大降低开发的效率。如果使用vue-cli，我们可以快速生成vue项目，而不需要我们一点点的去创建/管理项目所需要的各种文件夹/文件。vue-cli是对webpack的再封装，超级webpack。

2.vue-cli的优势

  1. 简单快速

    使用vue-cli，我们可以快速的新建一个vue项目，甚至可以使用图形化界面进行新项目的创建。

  2. 功能丰富

    vue-cli支持Bable，Ts，ESLint，Axios等。

  3. 易于扩展

    vue-cli有很多插件，可以实现大部分我们想要的功能。

3.如何使用vue-cli

1.安装vue-cli

1
2

npm install -g @vue/cli
vue -V

2.创建vue项目

1

vue creat 项目名

执行这条命令后，会进入一个交互式的面板，会让你选择项目需要的插件，以及各种配置，具体配置根据项目和个人使用习惯而有所区别，我第一次创建项目使用的配置是：

  1. pick a preset –>manually select features
  2. check the features needed for your project –>bable，router，Linter
  3. use history mode for router –>n
  4. pick a linter config –>ESLint + Standard config
  5. pick addition lint features –>Lint to save
  6. where do you prefer placing config for … –>In dedicated config files
  7. save this as a present for future projects –>n

3.配置项目

创建项目之后，我们可能会想要进行一些自定义配置，比如指定端口号，自动打开页面等，进行这些有两种方式，一种是在package.json中配置，另一种是新建vue.config.js进行配置，这里推荐第二种。

5.项目目录说明

  1. node_modules

    项目依赖包。

  2. public

    存放静态资源。

    index.html：

    是一个模板文件，作用是生成项目的入口文件。浏览器访问项目的时候就会默认打开的是生成好的index.html。

  3. src 项目核心文件

    1. assets

      样式类文件，如css、less、sass以及外部的js文件都放在这里。

    2. components

      存放自定义组件的文件夹。

    3. App.vue

      根组件，所有页面都是在App.vue下进行切换的，也可以理解为所有的路由也是App.vue的子组件。

    4. main.js

      入口文件，主要作用是初始化vue实例，同时可以在此文件中引用某些组件库或者全局挂载一些变量。

  4. .babelrc

    babel配置参数。

  5. .gitignore

    git忽略文件。

  6. package-lock.json

    是在npm install时候生成的一份文件，用于记录当前状态下实际安装的各个npm package的具体来源和版本号。

  7. package.json

    模块基本信息项目开发所需要的模块，版本，项目名称。

  8. README.md 项目说明

  9. webpack.config.js webpack配置文件

6.Element UI

1.ElementUI简介

Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

组件库是一组封装好的组件，我们可以直接使用组件库里定义好的各种组件来实现我们项目的功能，节省了我们自己定义各种组件样式的过程，大大提高了开发效率。

ElementUI手册：https://element.eleme.cn/2.0/#/zh-CN/component/installation

6.组件化开发

1.什么叫组件化开发

在使用vue-cli进行开发时，使用组件化开发方式，也就是.vue文件。一个.vue文件就是一个组件，里面包含了html，js，css三部分，通过这种方式，分离了每个组件之间的关联，提高了内聚，因此叫做组件化开发。

2.新建组件

vue-cli的组件都写在src/components里面，如果我们需要新建组件，可以在webstorm里直接右键选择新建vue components，会自动生成components的结构，很快很方便。
