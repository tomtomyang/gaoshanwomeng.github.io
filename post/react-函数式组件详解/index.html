React中有两种定义组件的方式：

  1. 函数式定义组件

    1function Welcome() {
2  return <h1>函数式</h1>
3}

  2. 类式定义组件

    1class Welcome extends React.Component {
2  render() {
3    return <h1>类式</h1>;
4  }
5}
