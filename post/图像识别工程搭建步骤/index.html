1. 数据准备

一般分为 5 个步骤

  1. 拍摄照片或者下载数据集；

  2. 图像标注；

  3. 图像数据处理：

图像合成、图像裁剪、改变图像像素大小；

  1. 检查标注文件的正确性；

  2. 划分训练集、验证集；

2. 搭建神经网络

3. 训练模型

读取数据的多线程生成器，多 gpu 并行训练、保存模型。

4. 模型评估和测试

加载模型，使用实际场景中的图像数据测试模型效果，单张图片检测，实时视频检测。

5. 模型部署

开发 web 服务端接口，开发 web 前端页面。
