1.3 控制台侧边栏

单击“显示控制台边栏” 以显示边栏，这对于过滤非常有用。

1.4 控制台设置

  1. Hide network messages隐藏网络请求信息

  2. Persist messages across page loads跨页面时清除控制台信息

  3. Filter out messages from different contexts过滤不用上下文消息

    假设您的网页上有一个广告。广告嵌入在<iframe>中并在您的控制台中生成大量消息。由于此广告处于不同的JavaScript上下文中，因此隐藏其消息的一种方法是启用仅所选上下文复选框。

  4. message grouping消息分组

  5. Log XHR and Fetch requests记录出所有的xml和fetch请求

  6. Eager Evaluation尽早求值

    控制台中键入JavaScript表达式时，“Eager Evaluation”将显示该表达式的返回值的预览

  7. autocomplete from history自动填充

    键入表达式时，控制台的自动完成弹出窗口会显示您之前运行的表达式

2. elements面板

2.1 查看dom节点

2.2 搜索dom节点

按Control+F或Command+ F（Mac）。搜索栏在DOM树的底部打开

2.3 编辑dom节点

在DOM树中，双击要修改的内容，文本突出显示为蓝色，表明已被选中。

2.4 重排dom节点

在DOM树中，选中元素进行拖拽。

2.5 模拟dom节点状态

可以通过:hov可以强制节点留在美国一样:active，:hover，:focus，:visited，和:focus-within等。

2.6 隐藏dom节点

选中dom节点，按h键，可以隐藏节点。

其本质也是添加通过添加类名__web-inspector-hide-shortcut__，使用visibility: hidden !important;。

2.7 删除dom节点

选中dom节点，按delete键，可以隐藏节点。
