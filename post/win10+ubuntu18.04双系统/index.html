win10+ubuntu18.04双系统
Tomtom Young 发布于2020-05-26 更新于2022-06-29 共743字 阅读2分钟

1.下载 ubuntu 系统

ubuntu 官网下载镜像： https://ubuntu.com/download/desktop

目前 ubuntu20.04LST 版本已经发布，但安装过程和 18.04LST 应该没有什么不同。

2.制作启动 U 盘

ubuntu 系统镜像很小，找一个不用的 U 盘，使用 UltralISO 或 Rufus 制作启动盘

1. 加载系统镜像

菜单栏 文件->打开 ，选择 ubuntu18.04 镜像文件

2. 写入 U 盘

菜单栏->启动->写入硬盘映像，选择对应 U 盘 写入方式选择 USB-HDD+，选择便捷启动->写入新的驱动器引导扇区->syslinux，这个便捷启动应该是必选，否则 U 盘不会识别硬盘，无法安装系统。

3. 制作完成

3.创建硬盘空白分区

使用 windows 的磁盘工具划分出 50G 以上的空白硬盘空间，我划分了 80G，安装软件多了之后感觉不够用了，如果空间足够大，建议划分出 100G 给 ubuntu。

4.设置启动模式

1. 关闭快速启动

快速启动设置在 win10->电源选项->选择电源按钮的功能->更改当前不可用设置->关机设置->关闭快速启动

2. 关闭安全启动

安全启动在 bios 里设置 secure boot 为 disabled

5.其他设置

  1. 在 bios 里将启动设置为 UEFI
  2. 关闭独显！关闭独显！关闭独显！否则进入 ubuntu 系统后会黑屏卡住

6.安装 ubuntu

  1. 进入 u 盘启动，选择直接 install 或者预览

  2. 其他步骤都可以根据自己的情况安装，强力推荐最小安装方式，没有乱七八糟的软件

  3. 进入分区界面，选择其他选项

  4. 选择自己分好的空闲分区，将空闲分区分为 3 个部分 /，swap， /home

    这 3 个分区分别是：

    1. /：>20G 用于存放各种系统文件

    1. swap：大于自己电脑的内存，我的笔记本电脑是站位 16G 内存，就分出 16G 空间给 swap

    1. /home：剩余所有空间 用于存放用户的各种文件

    1. 很多教程需要开辟出 200M 的空间给/boot 分区，据我查资料看，现在已经不需要单独的/boot 分区来引导启动，如果开辟还会造 成空间的浪费和磁盘空间支离破碎

  5. 等待安装完成

  1. 恢复安全启动，打开独显，进入 ubuntu 进行配置
