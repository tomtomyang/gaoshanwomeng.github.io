1.什么是React Native

要了解RN，肯定是要从官网入手：

按照官网的介绍，RN是使用React来创建Android和ios的原生应用的JavaScript框架。

RN具有以下的特点：

  1. 一次学习，随处编写

    因为RN使用的都是React的语法，对于前端人员可以快速上手。

  2. 酌量添加，多少随意

    随时都可以把RN无缝集成到你已有的Android或iOS项目。

  3. 流水的多平台，铁打的React

    绝大多数情况下，多个平台都共享一份代码，使用通用的技术React。

2. React Native基本原理

React Native提供了一些平台无关的抽象核心组件，像View、Text、Image等，可以直接映射渲染为对应平台的原生UI组件。
